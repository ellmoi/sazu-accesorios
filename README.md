# Sazu Accesorios

Demostración de tienda multicategoría creada únicamente con HTML5, CSS3 y JavaScript Vanilla.

## Uso

Abre `index.html` directamente. Para ver la administración, abre `admin.html`.

Incluye 18 productos, 8 clientes, 10 pedidos, 5 niveles de clientes, 3 descuentos mayoristas, 3 estados de inventario y 7 estados de pedido. El carrito, la sesión y las órdenes nuevas usan `localStorage`. Los pagos son simulados.
